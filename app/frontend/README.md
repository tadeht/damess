# Frontend - Hệ thống quản lý yêu cầu nội bộ

## 1. Công nghệ

- React
- Vite
- Tailwind CSS
- React Router
- Axios
- Recharts
- Lucide React

## 2. Cài đặt

Yêu cầu máy đã cài Node.js.

```bash
npm install
```

Tạo file `.env` từ `.env.example`:

```bash
cp .env.example .env
```

## 3. Chạy frontend

```bash
npm run dev
```

Frontend mặc định chạy tại:

```text
http://localhost:5173
```

Backend mặc định:

```text
http://localhost:5000/api
```
