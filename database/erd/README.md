# ERD - Hệ thống quản lý yêu cầu nội bộ

File ERD chính:

- `erd.mmd`

File này dùng cú pháp Mermaid ER Diagram. Có thể xem bằng:

- Mermaid Live Editor.
- Extension Markdown/Mermaid trong VS Code.
- Công cụ vẽ tài liệu hỗ trợ Mermaid.

## Ghi chú

ERD hiện tại phục vụ mức triển khai 2, tập trung vào các bảng cốt lõi:

- roles
- users
- departments
- request_types
- priorities
- statuses
- requests
- request_comments
- request_histories

Bảng `attachments` chưa đưa vào ERD mức 2 để giảm phạm vi triển khai. Nếu còn thời gian, có thể bổ sung sau.
